# Vault-Tec Terminal Network — Version 1

This ZIP is the full site package for **Version 1**.

## Highlights
- Fallout-style terminal theme with multiple phosphor palettes
- News / Projects collections driven by JSON + Markdown
- Tools pages for passwords, downloads, gallery, and contact notes
- Tube Status page with Underground / Elizabeth line / DLR / Overground live status
- Departure Boards page with station search, summary train prediction, and detailed train prediction

## Configuration
Edit:

```js
/assets/js/config/transport-config.js
```

Add your TfL `app_key` and (optionally) Tickernet key there, then hard-refresh the site.
